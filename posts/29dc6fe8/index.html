Docker常用命令

docker 基础命令

  • 启动docker
  • 关闭docker
  • 重启docker
  • 查看docker 运行状态 ——如果是在运行中 输入命令后 会看到绿色的active
  • 查看docker 版本号信息
  • 查看docker 详细信息 ——–此命令可以查看到docker 中容器运行个数 以及镜像个数等等
  • 设置开机启动
  • 关闭开机启动

docker 镜像命令

  • 查看自己服务器中docker 镜像列表
  • 拉取镜像 不加tag(版本号) 即拉取docker仓库中 该镜像的最新版本latest 加:tag 则是拉取指定版本
  • 运行镜像
  • 删除镜像 ——当前镜像没有被任何容器使用才可以删除

docker 容器命令

  • 查看运行中的所有容器
  • 查看正在运行容器列表
  • 停止容器
  • 重启容器
  • 启动容器
  • kill 容器
  • 进入容器
  • 退出容器

docker 网络命令

  • 列所有列表的网络
  • 创建macvlan网络
